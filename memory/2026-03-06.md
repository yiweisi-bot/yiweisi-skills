- "截图分享"、"发朋友圈"

**默认简洁版（手机文本版）**：
- 不包含上述关键词时 → 自动使用手机版
- 纯文本 + Markdown 格式，无 ASCII 边框
- 在小屏幕上完美显示

#### 3. 技术实现

- SKILL.md 明确关键词判断逻辑
- 模型根据用户输入自动选择参数

#### 4. 测试结果
- ✅ 默认手机版：简洁清晰，适合小屏幕
